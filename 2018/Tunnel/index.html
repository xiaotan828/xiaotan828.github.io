As President Xi came into power, the Internet access in China has become more and more restrictive. Most foreign websites are blocked in mainland China under the country’s policy of Internet censorship, which include Google, Youtube, Facebook, Twitter, Wikipedia (CN site) and so on. To visit these websites, you need to break through blockade of GFW.

What is shadowsocks?

Shadowsocks is an open-source encrypted proxy project, widely used in mainland China to circumvent Internet censorship. It was created in 2012 by a Chinese programmer named “clowwindy”, and multiple implementations of the protocol have been made available since. Typically, the client software will open a socks5 proxy on the machine it is run, which internet traffic can then be directed towards, similarly to an SSH tunnel. Unlike an SSH tunnel, shadowsocks can also proxy UDP traffic. There are several different types of shadowsocks now but all of them are based on socks5 protocol. (Read more in Wikipedia)

What is the different between VPN and SS?

Technically, shadowsocks has nothing to do with VPN. It’s not a VPN, it’s a proxy. VPN is a virtual network interface which you can send everything through it. Proxy is a server used to forward data to another. There are two mainly difference between them.

  1. To use a proxy, you had to let program know there is a proxy. But you don’t have to config programs to use VPN.
  2. You can transfer any kind of data through VPN. With proxy, only TCP can be delivered. (actually, in some version of shadowsocks, UDP also works)

(Read more in this Quora)

Why is a VPS needed?

Look at the following picture. In the practical use, encrypted data is transferred through the VPS(SS Server) so that the GFW doesn’t know where is the data from, and will naturally allow the data to pass through.

  • Make sure these websites are accessible at the location of your VPS.
  • Make sure the IP address of the VPS haven’t been blocked by the GFW.
  • The shorter the distance between you location and the location of your VPS, the faster your Internet will be.
